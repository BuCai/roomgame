# 棋牌房卡游戏

### 介绍
房卡类棋牌游戏
游戏大厅,斗地主,3D麻将,跑得快等益智棋牌游戏,支持开房模式，支持茶馆俱乐部。

### 软件架构
#### 开发工具：Unity3D
#### 开发语言：Java + C#
#### 服务端框架：SmarkfoxServer2x + MySQL
#### 客户端语言：C#
